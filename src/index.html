Getting Started

Import an image by clicking the Import button or by dragging and dropping an image onto the canvas area.

Dithering Styles

  • Bayer-Ordered - Classic ordered dithering with control over matrix size
  • Floyd-Steinberg - Error diffusion dithering, great for black and white
  • Color Floyd-Steinberg - Error diffusion for color images
  • Random-Ordered - Adds noise-based texture to dithering
  • Atkinson - Apple's classic dithering algorithm, good for retro look
  • Bit Tone - Creates bit-crushed tone effect

Adjustments

  • Scale - Controls the intensity of the dithering effect
  • Contrast - Adjusts image contrast before dithering
  • Midtones - Affects the middle values of the image
  • Highlights - Adjusts the brightest areas
  • Luminance Threshold - Sets the cutoff point for black vs. white

Keyboard Shortcuts

  • Ctrl+Z - Undo
  • Ctrl+Y - Redo
  • + - Zoom in
  • - - Zoom out
  • 0 - Reset zoom
  • Ctrl+Shift+/ - Show this help

About the Project

DitherDemon is a web-based image processing tool that specializes in applying various dithering techniques to images. Dithering is a technique used in computer graphics to create the illusion of color depth in images with a limited color palette.

This application was created to explore the artistic potential of dithering and to provide a simple, accessible tool for artists, designers, and enthusiasts to experiment with these techniques.

Technical Details

DitherDemon is built with vanilla JavaScript and HTML Canvas. No external libraries were used for the image processing algorithms - everything is custom-coded for optimal performance and maximum control over the dithering process.
